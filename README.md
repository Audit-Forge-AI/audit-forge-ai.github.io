# AuditForge AI

AuditForge AI is a free website auditing platform that helps developers, founders, and businesses identify issues affecting their websites.

It analyzes websites across multiple categories, including SEO, accessibility, performance, security, content quality, and site structure, providing actionable insights to improve overall quality and visibility.

## Features

- 🔍 SEO auditing
  - Title and meta analysis
  - Heading structure checks
  - Canonical and robots validation
  - Structured data review

- ♿ Accessibility analysis
  - Alt text coverage
  - Form label checks
  - ARIA validation
  - Focus and usability considerations

- ⚡ Performance insights
  - Asset weight analysis
  - Render-blocking resource detection
  - Image optimization checks
  - Cache recommendations

- 🔒 Security reviews
  - HTTPS verification
  - Security header checks
  - Mixed-content detection
  - Exposure of common development artifacts

- 📝 Content evaluation
  - Thin content detection
  - Duplicate content checks
  - Broken link identification
  - Readability assessment

- 🗺️ Site structure analysis
  - Crawl depth insights
  - Orphan page detection
  - Sitemap validation
  - URL hygiene checks

## Why AuditForge AI?

Many website auditing tools hide important features behind expensive subscriptions. AuditForge AI aims to make essential website analysis accessible to everyone through a simple and transparent experience.
